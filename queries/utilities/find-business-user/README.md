# Find BusinessUserId

**Category**: Utility
**Created**: 2026-04-26

## Purpose

Looks up a `BusinessUserId` by searching the Person table by name. Useful when you need to find your own BusinessUserId for testing queries that require it (e.g. ReportFavourites).

## Usage

Change the `LIKE '%Haseeb%'` filter to match the person you're looking for.

## Known IDs

| Name | BusinessUserId | HomeSiteId |
|------|---------------|------------|
| Abdul Haseeb | 317646 | 3187 |

## Tables Used
- `BusinessUser` (Access_CS)
- `Person` (Access_CS)
